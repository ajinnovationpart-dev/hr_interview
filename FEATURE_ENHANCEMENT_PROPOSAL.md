# 면접 관리 시스템 부가 기능 제안서

## 📋 개요

현재 시스템에 추가할 수 있는 부가 기능들을 우선순위와 구현 난이도에 따라 분류하여 제안합니다.

---

## 🎯 핵심 기능 (우선순위 높음)

### 1. 이력서 업로드 및 관리 ⭐⭐⭐

**기능 설명:**
- 면접 등록 시 지원자별 이력서 파일 업로드
- 면접관 포털에서 해당 면접 시간에 맞춰 이력서 조회
- 이력서 다운로드 및 미리보기

**구현 내용:**
- **백엔드:**
  - 파일 업로드 API (`POST /api/candidates/:id/resume`)
  - 파일 저장소 (OneDrive/SharePoint 또는 로컬 스토리지)
  - 이력서 조회 API (`GET /api/candidates/:id/resume`)
  - 면접관별 이력서 접근 권한 관리
  
- **프론트엔드:**
  - 면접 등록 페이지에 이력서 업로드 필드 추가
  - 면접관 포털에 이력서 조회 섹션 추가
  - PDF/이미지 미리보기 기능
  - 이력서 다운로드 버튼

**데이터 구조:**
```typescript
interface Candidate {
  candidate_id: string;
  name: string;
  email: string;
  phone: string;
  position_applied: string;
  resume_url?: string;        // 이력서 파일 경로
  resume_file_name?: string;  // 원본 파일명
  resume_uploaded_at?: string; // 업로드 일시
  cover_letter_url?: string;   // 자기소개서 (선택)
  // ... 기존 필드
}
```

**면접관 포털 UI:**
- 면접 일정 확인 페이지에 "이력서 보기" 버튼 추가
- 면접 시간 전/후 이력서 조회 가능
- 이력서 미리보기 (PDF 뷰어 또는 이미지)

---

### 2. 면접 평가 및 평점 시스템 ⭐⭐⭐

**기능 설명:**
- 면접 완료 후 면접관이 지원자 평가 입력
- 항목별 평점 (기술력, 커뮤니케이션, 적합도 등)
- 종합 평가 및 합격/불합격 결정

**구현 내용:**
- **평가 항목:**
  - 기술 역량 (1-5점)
  - 커뮤니케이션 능력 (1-5점)
  - 업무 적합도 (1-5점)
  - 팀워크 (1-5점)
  - 종합 평가 (1-5점)
  - 합격/불합격 여부
  - 평가 코멘트

- **데이터 구조:**
```typescript
interface InterviewEvaluation {
  evaluation_id: string;
  interview_id: string;
  candidate_id: string;
  interviewer_id: string;
  technical_score: number;      // 1-5
  communication_score: number;   // 1-5
  fit_score: number;             // 1-5
  teamwork_score: number;        // 1-5
  overall_score: number;         // 1-5
  recommendation: 'PASS' | 'FAIL' | 'CONSIDER';
  comments: string;
  strengths: string[];           // 강점
  weaknesses: string[];          // 보완점
  created_at: string;
}
```

- **UI:**
  - 면접 완료 후 평가 입력 폼
  - 평가 결과 대시보드 (평균 점수, 합격률 등)
  - 면접관별 평가 비교

### 4. 면접 결과 관리 및 채용 파이프라인 ⭐⭐⭐

**기능 설명:**
- 지원자별 채용 단계 추적
- 합격/불합격 결정 및 통보
- 제안서 발송 및 입사 일정 관리

**구현 내용:**
- 채용 단계: 지원 → 서류심사 → 1차 면접 → 2차 면접 → 최종 면접 → 제안 → 입사
- 각 단계별 상태 관리
- 자동 이메일 통보 (합격/불합격)
- 입사 예정일 관리

## 📝 구현 현황 요약

### ✅ 구현 완료
- **이력서 업로드·관리**: 면접 등록 시 지원자별 업로드, 면접관 포털 이력서 다운로드
- **면접 평가·평점**: 면접관 포털 평가 입력(별점/합격여부/코멘트), 관리자 면접 상세에서 평가 현황 조회
- **통계에 평가 반영**: 통계 및 리포트 페이지에 평가 통계(합격률, 평균 점수) 카드 표시
- **지원자 포트폴리오**: 지원자 관리·상세에 포트폴리오 URL 필드 추가
- **면접 노트**: 평가 코멘트 필드를 면접 노트로 활용 (라벨 및 placeholder 안내)
- **채용 파이프라인**: 지원자 상태(지원/서류심사/면접/제안/불합격 등)는 지원자 관리에서 이미 사용 중

### 🔲 미구현·추가 예정
- 면접 리포트 PDF 자동 생성
- 캘린더 연동, 사전 질문지, 면접 피드백 시스템 등 Phase 4
