# 로컬 개발 환경 설정 가이드

## 문제: ERR_CONNECTION_REFUSED

면접관 정보를 불러오지 못하는 문제는 백엔드 서버가 실행되지 않았거나 API URL이 잘못 설정되었을 때 발생합니다.

## 해결 방법

### 1. 백엔드 서버 시작

터미널에서 다음 명령어를 실행하세요:

```bash
cd backend
npm run dev
```

서버가 정상적으로 시작되면 다음과 같은 메시지가 표시됩니다:
```
Server is running on port 3000
```

### 2. 프론트엔드 환경 변수 설정

`frontend` 폴더에 `.env.local` 파일을 생성하고 다음 내용을 추가하세요:

```bash
VITE_API_URL=http://localhost:3000/api
```

**파일 경로:** `frontend/.env.local`

### 3. 프론트엔드 서버 재시작

프론트엔드 서버를 중지(Ctrl+C)하고 다시 시작하세요:

```bash
cd frontend
npm run dev
```

### 4. 확인

브라우저에서 http://localhost:5173 접속하여 정상 작동하는지 확인하세요.

## 현재 설정 확인

현재 API URL이 `http://192.10.10.76:3000/api`로 설정되어 있다면, 로컬 개발 환경에서는 `http://localhost:3000/api`로 변경해야 합니다.

## 문제 해결 체크리스트

- [ ] 백엔드 서버가 포트 3000에서 실행 중인가?
- [ ] `frontend/.env.local` 파일이 존재하고 올바른 URL이 설정되어 있는가?
- [ ] 프론트엔드 서버가 재시작되었는가?
- [ ] 브라우저 콘솔에 에러가 없는가?

## 추가 도움말

- 백엔드 서버가 시작되지 않으면 `backend/.env` 파일을 확인하세요.
- 포트 3000이 이미 사용 중이면 다른 포트로 변경하거나 기존 프로세스를 종료하세요.
