# 배포 후 테스트 방법 (로컬)

## 1. 최신 코드 받기

```bash
cd E:\hr-sample
git pull origin main
```

## 2. 프론트엔드 실행

```bash
cd frontend
npm install
npm run dev
```

브라우저에서 **캐시 없이** 새로고침(Ctrl+Shift+R) 또는 시크릿 창으로 접속합니다.

## 3. 면접 시간 선택 확인

- **면접 등록** 페이지 → "2. 면접 일시" → **시작 시간** 클릭
  - 기본값 **09:00**으로 표시되는지
  - 시간 목록에서 **00~08시, 12시, 18~23시**가 비활성화(회색)인지 확인
- 비활성화가 보이지 않으면 브라우저 강력 새로고침 후 다시 시도

## 4. 배포 서버(Vercel/Netlify 등) 사용 시

- 저장소 연결이면 `main` 푸시 후 자동 배포됩니다.
- 변경이 안 보이면 해당 서비스에서 **Redeploy** 실행 또는 캐시 무효화 후 확인하세요.
