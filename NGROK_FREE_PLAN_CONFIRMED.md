# ngrok 무료 플랜 제한 확인

## 확인된 사실

**ngrok 무료 플랜은 동시에 1개의 ngrok 클라이언트 세션만 지원합니다.**

- ❌ 2개 터널 동시 실행: 불가능
- ✅ 1개 터널만 실행: 가능

## 현재 상황

- 기존 3030 포트 ngrok이 실행 중
- 3000 포트 ngrok을 추가로 실행하려고 함
- **무료 플랜 제한으로 불가능**

## 해결 방법

### 옵션 1: 3000 포트만 실행 (권장)

기존 3030 포트 ngrok을 종료하고 3000 포트만 실행:

```bash
# 기존 ngrok 종료 (3030 포트)
.\stop-ngrok.ps1

# 3000 포트만 실행
ngrok http 3000
```

### 옵션 2: 3030 포트 유지

기존 3030 포트 ngrok을 그대로 두고, 3000 포트는 실행하지 않음

### 옵션 3: 유료 플랜 업그레이드

여러 터널을 동시에 사용하려면 ngrok 유료 플랜 필요:
- https://ngrok.com/pricing

## 권장 사항

**Backend 서버(3000 포트)만 터널링하는 것을 권장합니다:**

1. GitHub Pages에서 Backend API에 접근해야 함
2. 3000 포트가 더 중요함
3. 필요시 나중에 3030 포트로 전환 가능

## 빠른 실행

```bash
# 1. 기존 ngrok 종료
.\stop-ngrok.ps1

# 2. 3000 포트만 실행
ngrok http 3000

# 3. 터널 URL 확인
# http://127.0.0.1:4040
```

## 참고

- 무료 플랜: 1개 세션만 지원
- 유료 플랜: 여러 세션 동시 지원
- 현재는 3000 포트만 실행하는 것이 가장 현실적
